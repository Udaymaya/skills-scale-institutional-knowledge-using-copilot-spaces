# OctoAcme — Stakeholder Communication Template

## Purpose
Provide reusable templates for consistent, clear stakeholder communication across project phases. Adapting these templates ensures stakeholders receive timely, accurate information and reduces miscommunication.

## Notes
- Tailor templates to the audience — executives may need summary-only versions.
- Archive all stakeholder communications in the project's shared channel or repo.
- Review communication frequency with stakeholders at the start of each phase.
